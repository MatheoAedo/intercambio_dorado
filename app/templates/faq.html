Preguntas frecuentes

Respuestas rápidas sobre el funcionamiento de Intercambio Dorado. Diseñado para ser fácil de usar, especialmente para personas adultas mayores.

Cuenta y acceso

¿Necesito una cuenta para usar la plataforma?

Puedes ver la información general sin cuenta, pero para publicar servicios, crear intercambios o valorar, necesitas iniciar sesión.

¿Cómo inicio sesión?

Presiona el botón “Iniciar sesión”, escribe tu correo y contraseña, y luego presiona Entrar. Si te equivocas, puedes intentarlo nuevamente.

¿Qué pasa si olvido mi contraseña?

En esta versión de demostración (mock) se usan cuentas de prueba. En la versión final se agregará recuperación de contraseña y soporte.

¿Hay un perfil para administrador?

Sí. El administrador puede revisar el funcionamiento del sistema y gestionar información para mantener el orden y la seguridad.

Servicios

¿Qué tipo de servicios se pueden publicar?

Se pueden publicar servicios de apoyo cotidiano: ayuda con celular, acompañamiento, trámites, orientación básica, apoyo comunitario, pequeñas tareas del hogar, etc. No se permiten servicios riesgosos, ilegales o que pongan en peligro a las personas.

¿Puedo publicar un servicio si soy adulto mayor?

Sí. El sistema está pensado para que cualquier persona pueda participar. Un adulto mayor puede publicar servicios (por ejemplo: enseñanza de cocina, acompañamiento, apoyo emocional, manualidades, etc.).

¿Cuántos créditos por hora se puede cobrar?

En esta demo se controla visualmente entre 1 y 10 créditos/hora para mantener el intercambio justo. En la versión final se podrán definir rangos según el tipo de servicio.

¿Qué significa “créditos”?

Los créditos son una forma simple de representar el valor del tiempo y el apoyo entre personas. No es dinero: es un sistema de intercambio para promover colaboración.

¿Puedo editar o eliminar un servicio?

Sí. El usuario puede editar/eliminar sus propios servicios (en la versión con base de datos se validará por usuario). El administrador puede gestionar todos para mantener el control del sistema.

¿Qué información debo escribir en la descripción?

Se recomienda escribir: qué ofreces, cuánto dura, qué necesitas para realizarlo y cualquier detalle importante. Mientras más claro, más fácil será para otros entender tu servicio.

Intercambios

¿Qué es un intercambio?

Un intercambio es el registro de una solicitud o acuerdo por un servicio publicado. Permite llevar control de fecha y estado.

¿Qué estados puede tener un intercambio?

Pendiente, confirmado, en progreso, completado o cancelado. Esto permite saber claramente en qué etapa se encuentra.

¿Cómo se crea un intercambio?

Se selecciona un servicio, se asigna fecha y estado. Luego el administrador o el usuario puede actualizarlo según corresponda.

¿Puedo cancelar un intercambio?

Sí. Si un intercambio no se puede realizar, se puede marcar como cancelado. Esto ayuda a mantener el historial ordenado.

¿Cómo sé si mi intercambio fue aceptado?

En esta versión mock, el estado se actualiza manualmente. En la versión final se podrán agregar notificaciones y confirmaciones.

Valoraciones y reputación

¿Cómo se valora un servicio?

La valoración es de 1 a 5 estrellas con comentario opcional. Sirve para construir confianza y reputación.

¿Para qué sirven los comentarios?

Los comentarios ayudan a que otros usuarios conozcan la experiencia real. También sirven para mejorar la calidad del servicio.

¿Cómo se calcula la reputación?

La reputación se calcula con el promedio de puntuaciones por servicio y el total de valoraciones recibidas. Mientras más valoraciones positivas, mayor confianza genera.

¿Puedo cambiar una valoración?

En esta demo sí se puede editar. En la versión final se definirán reglas claras para evitar abusos y mantener la transparencia.

Seguridad y ayuda para adultos mayores

¿Es seguro usar esta plataforma?

El objetivo del sistema es promover confianza mediante reputación, control de roles y registro de actividades. Además, el administrador puede supervisar el contenido.

¿Qué hago si no entiendo algo?

No te preocupes. La plataforma está pensada para ser simple. Puedes revisar este FAQ o pedir ayuda a un familiar. En la versión final se incluirá soporte y guías paso a paso.

¿Qué recomendaciones hay para evitar estafas?

Recomendaciones básicas: no compartir claves, no entregar datos bancarios, revisar reputación, y preferir intercambios claros y confirmados.

¿La letra se puede ver más grande?

Sí. Se está priorizando accesibilidad con texto legible, botones grandes y contraste. También puedes usar el zoom del navegador (Ctrl + +).

¿Mis datos están protegidos?

En la versión final se implementarán medidas como contraseñas encriptadas, validaciones y control de acceso por roles.
